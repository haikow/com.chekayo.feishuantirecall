# 飞书防撤回 (Feishu Anti-Recall) · LSPosed 模块

让被撤回的飞书消息原文继续显示 —— **包括你从未打开过的聊天**。

> ⚠️ 本仓库仅提供编译好的 APK 下载，不包含源码。仅供学习研究与个人使用，使用风险自负。

---

## 📦 支持范围

| 项 | 说明 |
|---|---|
| 架构 | 仅 arm64-v8a（覆盖绝大多数手机） |
| 环境 | Android，需 Root + LSPosed（或兼容的 Xposed 框架） |
| 飞书 | 在 7.52.4 上验证；其它版本一般可用，若失效需等待更新 |

## 🔧 安装与使用

1. 在 [Releases](../../releases) 下载最新 APK 并安装。
2. 打开 **LSPosed**，启用本模块。
3. 勾选作用域：**飞书**。
4. **强制停止飞书**后重新打开（设置 → 应用 → 飞书 → 强行停止），即可生效。

## 📖 说明

- 模块启用后收到、随后被撤回的消息，打开聊天即可看到原文。
- 会话里仍会显示「对方撤回了一条消息」的提示行，属正常现象。
- 模块启用之前就被撤回的旧消息无法恢复。
- 飞书大版本更新后若失效，请等待模块更新。
